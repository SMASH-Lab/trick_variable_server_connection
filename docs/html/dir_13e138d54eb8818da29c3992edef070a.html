file  test01_set_one_reading.c
 This is a simple test that shows how to connect and interact to a Trick Variable Server for querying and reading simulation data (just one reading). The source simulation is the SIM_cannon_jet reported in the Trick Tutorial "Trick Simulation Environment User Training Materials Trick 2013.0 Release", see Section 9.0 and subsequent. The program takes as first input parameter the port number on which the Trick Variable Server is active. The server is supposed to run locally. If not, the IP address must be provided after the port number as the second input parameter.
